mysqli_sql_exception クラス

(PHP 5)

導入

mysqli の例外を処理するクラスです。

クラス概要

mysqli_sql_exception extends RuntimeException {
/* プロパティ */
protected $code ;
protected $sqlstate ;
}

プロパティ

message

エラーメッセージ。

file

エラーが発生したファイル。

line

エラーが発生した行番号。

code

エラーの原因コード。

sqlstate

エラーの SQLSTATE。
